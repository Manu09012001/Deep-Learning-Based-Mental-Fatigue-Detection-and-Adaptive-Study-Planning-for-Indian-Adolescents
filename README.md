# Deep Learning-Based Temporal Transformer for Mental Fatigue Detection using Study Behaviour Time-Series Data for Indian Adolescent Population

## Overview

Mental fatigue among adolescents significantly affects learning efficiency, concentration, and academic performance. This project proposes a Temporal Transformer-based deep learning framework that analyzes study behaviour time-series data to detect mental fatigue and generate personalized adaptive study plans.

The system processes temporal behavioural data such as study duration, break frequency, sleep duration, screen time, and attention-related indicators to classify fatigue into Low, Moderate, and High levels. Based on the predicted fatigue level, an adaptive study planner recommends personalized study schedules, break intervals, and subject priorities.

---

## Objectives

- Detect mental fatigue using study behaviour time-series data.
- Develop a Temporal Transformer-based deep learning model.
- Classify fatigue into Low, Moderate, and High levels.
- Generate adaptive study plans.
- Improve learning efficiency among Indian adolescents.
